WordPress 6.3 發佈

2024-01-152023-08-09 作者: yungke
XFacebookLinkedInLineEmailTumblrMastodon

WordPress 6.3 是 WordPress 路線圖在第二階段的最後一個主要版本，在第二階段開發，主要集中在全站編輯器上，增加了其功能，並改進了界面和可用性。

你的網站沒有用到全站編輯器或佈景主題，其實是可以放心的升級 WordPress 6.3 核心。

在 WordPress 6.3 中，10 個版本的 Gutenberg 外掛被整合到核心中，如果你沒有在網站上使用 Gutenberg 外掛，你會在全站編輯器的界面中看到很多變化。

Kinsta WordPress 6.3 詳細說明

關於 WordPress 6.3 有很多的變化，我們引用 Kinsta 的官方文章說明，以及介紹 Dave Smith 製作的快速且有用的影片，該影片在短短 6 分鐘內解釋了 WordPress 6.3 的功能。

Dave Smith YouTube

WordPress 核心程式的更新是必然的，我們無法去阻止它，這也是為了網站安全及功能更齊全所必須這麼做。

這裡還有 WordPress 官方的影片，令人印象深刻。

令人印象深刻的 WordPress 6.3 landing page

希望利用 Kinsta 專業文章的講解和影片說明，讓你不要懼怕 WordPress 核心升級。
